# 新闻浏览器项目

一个类似浏览器的 Web 应用，用户可以在后端配置多个网站地址，在前端界面上直接点击打开查看。

## 功能特点

- ✅ 添加自定义网站链接
- ✅ 编辑已有网站信息
- ✅ 删除不需要的网站
- ✅ 一键点击在 iframe 中打开网站
- ✅ 美观的响应式界面（支持移动端）
- ✅ 实时加载和更新
- ✅ 移动端侧边栏切换
- ✅ 网站可访问性检测
- ✅ 超时自动提示并提供新窗口打开选项

## 技术栈

- **后端**: Node.js + Express
- **前端**: 原生 HTML/CSS/JavaScript
- **数据存储**: JSON 文件

## 安装步骤

1. 安装依赖
```bash
npm install
```

2. 启动服务
```bash
# 生产环境
npm start

# 开发环境（自动重启）
npm run dev
```

3. 访问应用
打开浏览器访问：http://localhost:3000

## 使用说明

### 添加网站
1. 在左侧"添加新网站"区域填写网站名称和地址
2. 点击"添加网站"按钮
3. 新增的网站会出现在网站列表中

### 访问网站
1. 在网站列表中点击任意网站卡片
2. 右侧会显示该网站的页面内容

### 编辑网站
1. 鼠标悬停在网站卡片上
2. 点击右上角的铅笔图标
3. 修改名称或地址

### 删除网站
1. 鼠标悬停在网站卡片上
2. 点击右上角的删除图标
3. 确认删除

## 数据存储

所有网站配置数据保存在 `data/urls.json` 文件中，您可以手动编辑此文件来批量修改配置。

## API 接口

- `GET /api/urls` - 获取所有网站列表
- `POST /api/urls` - 添加新网站
- `PUT /api/urls/:id` - 更新指定网站
- `DELETE /api/urls/:id` - 删除指定网站
- `GET /api/check-url?url=xxx` - 检查 URL 可访问性

## 注意事项

1. **iframe 限制**：某些网站可能禁止在 iframe 中显示（由于 X-Frame-Options 安全策略），这是正常的浏览器安全限制
2. **推荐添加的网站类型**：
   - 新闻网站（如人民网、新华网等）
   - 博客网站
   - 文档网站
   - 其他支持 iframe 嵌入的网站
3. **不建议添加的网站**：
   - 大型门户网站（如 Google、百度等通常禁止 iframe）
   - 社交网络（如 Facebook、微信等）
   - 银行和金融网站（安全考虑）
4. **移动端适配**：
   - 在移动设备上会自动切换到单列布局
   - 可以通过顶部按钮切换侧边栏显示
   - iframe 高度会自动适应屏幕
5. **URL 格式**：添加时会自动补全 http/https 前缀，无需手动输入
